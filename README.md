## Description

**VexorMotors Dealership API** is a REST API designed to serve as the backend core of a dealership management system, enabling secure and scalable management of vehicles, customers, inventory, and business operations.

The project is built on **Spring Boot 4 and Java 21**, following clean architecture principles, separation of concerns, and best development practices.

---

## Project Goals

- Centralize vehicle and customer management
- Facilitate purchase and sales operations
- Provide a secure foundation for future extensions
- Serve as backend for web or mobile applications

---

## Main Features

- Vehicle management
- Customer management
- Inventory control
- Purchase and sales operations
- Security and access control
- Automatic database migrations
